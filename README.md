# Rutas YoVoy - GUI

## Descripción
Este proyecto es una simulación visual de un sistema de transporte público desarrollado en Python con Tkinter.

## Características
- Interfaz gráfica profesional
- Uso de widgets: Label, Entry, Button, Frame, LabelFrame, Combobox, ScrolledText, Radiobutton, Checkbutton y Canvas
- Diseño basado en una paleta de colores coherente con el logotipo

## Nota
Este sistema es únicamente visual (no incluye lógica), como se solicita en el requerimiento.

## Ejecución
Ejecutar el archivo .py en Python para visualizar la interfaz gráfica.
